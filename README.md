# КВЕСТ: Геймифицированный ИИ-Планировщик Реальности

Мобильное приложение, превращающее повседневную рутину, привычки и долгосрочные цели в ролевую RPG-игру. Программа помогает поддерживать дисциплину, избегать выгорания с помощью микро-режимов выполнения задач (MCO) и автоматически генерирует сбалансированные планы достижения целей с помощью ИИ.

Оптимизировано для мобильных устройств с использованием **Capacitor** и собрано в готовый APK.

---

## 🎮 Основные игровые и функциональные механики

### 1. 🛡️ Прокачка Персонажа и RPG-система
* **Характеристики**: Развитие четырех ключевых сфер жизни при выполнении задач: **Тело** (`BODY`), **Дух** (`SPIRIT`), **Интеллект** (`INT`) и **Капитал** (`CAP`).
* **Опыт (XP) и Уровни**: За выполнение задач начисляется опыт. При накоплении 100 XP уровень персонажа повышается, полностью восстанавливая здоровье. Звание персонажа меняется от *Искателя приключений* до *Легенды квеста*.
* **Здоровье (HP)**: Запас здоровья персонажа равен 100. Если в течение дня не выполнить задачи, персонаж получает урон (-20 HP).
* **Стрики (Серии дней)**: Ежедневное выполнение квестов увеличивает серию («стрик») и дает множитель к получаемому опыту (до х1.5 за 5+ дней).

### 2. ⚡ Умные задачи (Ежедневные квесты)
* **Уровни сложности**: Легкие (10 XP), Средние (20 XP) и Сложные (30 XP) квесты.
* **Микро-режимы против выгорания (MCO — Micro-Commitment Option)**: Каждый квест имеет «облегченную» альтернативу (например, вместо 25 минут чтения — прочитать 1 страницу). Выполнение MCO-версии приносит меньше XP, но **сохраняет стрик активности** и защищает персонажа от урона по здоровью в конце дня.

### 3. 🏆 Эпические кампании (Долгосрочные цели)
* **Прогресс-бар целей**: Задачи с числовым целевым показателем (например, «прочитать 20 книг» или «опубликовать 30 постов»).
* **Контрольные точки (Milestones)**: Достижение промежуточных этапов (10%, 25%, 50%) дает крупные бонусы к опыту (+50 XP), а завершение кампании — супер-награду (+200 XP).

### 4. 🧠 Персональный ИИ-Геймдизайнер Реальности
Встроенный чат с ИИ-ассистентом для декомпозиции ваших целей в игровые квесты.
* **Оффлайн NLP-планировщик**: Если API-ключ не задан, приложение использует встроенные паттерны распознавания текста (спорт, языки, кодинг, блогинг) для мгновенной выдачи сбалансированного плана.
* **Интеграция с Google Gemini API**: В настройках можно ввести собственный API-ключ Gemini, чтобы ассистент генерировал индивидуальные, гибкие и креативные квесты в JSON-формате прямо под ваши запросы.

### 5. 🔔 Локальные уведомления
* Интеграция с нативным плагином `@capacitor/local-notifications`.
* Если у вас остались незакрытые квесты, приложение автоматически запланирует напоминание на **16:00**, призывая сохранить серию дней и защитить персонажа от потери здоровья.

### 6. 💾 Экспорт, Импорт и Резервные копии
* Хранение данных локально на устройстве в `localStorage`.
* Полный экспорт и импорт всего игрового профиля (данные персонажа, активные квесты, эпики, история чата) в один клик.

## 🛠️ Технологический стек

* **Frontend**: React 19, Vite.
* **Иконки**: Lucide React.
* **Нативные модули**: `@capacitor/core`, `@capacitor/local-notifications` (пуш-уведомления на мобильных устройствах).
* **Стилизация**: Адаптивный CSS с игровой темной темой и игровыми элементами.

## ⚙️ Инструкция по запуску

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки (в браузере)
```bash
npm run dev
```

### Синхронизация нативных плагинов (Capacitor)
```bash
npx cap sync
```

### Открытие проекта в Android Studio
```bash
npx cap open android
```
